# 需要安装的软件

1. yay(需要加入archlinuxcn源)
2. zsh(shell)
3. oh-my-zsh-git
4. powerline-fonts(字体)
5. base-devel（一些基础工具）
6. xorg（安装dwm依赖）
7. xorg-xinit（安装dwm依赖）
8. feh（壁纸设置）
9. udisks2（U盘管理）
10. udiskie（U盘管理）
11. pcmanfm(文件管理器)
12. archlinux-wallpaper(壁纸)
13. wqy-microhei(中文字体)
14. ttf-nerd-fonts-symbols（表情）
15. neofetch（总览）
