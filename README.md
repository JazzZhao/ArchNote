# 需要安装的软件

1. yay(需要加入archlinuxcn源)
2. zsh(shell)
3. base-devel（一些基础工具）
4. xorg（安装dwm依赖）
5. xorg-xinit（安装dwm依赖）
6. feh（壁纸设置）
7. udisks2（U盘管理）
8. udiskie（U盘管理）
9. pcmanfm(文件管理器)
10. archlinux-wallpaper(壁纸)
11. picom(美化半透明)
12. wqy-microhei(中文字体)
13. ttf-nerd-fonts-symbols（表情）
14. neofetch（总览）
